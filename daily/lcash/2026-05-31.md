### Session (18:36) [superwin]

**Context:** Debugging and fixing TAB scraper thrashing against Akamai WAF, plus investigating a TabTouch race count drop

**Key Exchanges:**
- TAB discovery was thrashing Akamai (~30 retries/hour) due to backoff code being bypassed — the 120s wrapper throws `asyncio.CancelledError` which isn't caught by `except Exception:`
- First backoff fix deployed but didn't work because the outer `_rediscovery_loop` in `app/adapters/tab.py` re-calls `_do_rest_poll` every 60s regardless, bypassing inner backoff
- Second fix moved backoff gating to the outer `_rediscovery_loop` level — confirmed working: 9 attempts in 3h11min vs ~95 previously (~10x reduction)
- TabTouch 240→92 race drop was transient (TabTouch.mobi server 500 + direct-mode timeout cascade), self-recovered to 134 within minutes
- curl_cffi probe: `www.tab.com.au` returns 200 with chrome131 fingerprint, but `api.beta.tab.com.au` returns 403 — API requires Akamai cookies set by SPA JavaScript

**Decisions Made:**
- Backoff ratchet: 1→5→15→30 min (capped) on consecutive failures — balances retry opportunity vs Akamai pressure
- Tier 2 interceptor wait reduced 60s→30s to fit within the 120s wrapper timeout budget (Tier 2 30s + Tier 3 15s + Tier 4 ~30s = 75s < 120s)
- DISCOVERY_INTERVAL kept at 3600s for steady-state (success path)
- Deferred CookieMinter build (4-6 hour session) — not auto-mode safe

**Lessons Learned:**
- `asyncio.CancelledError` is NOT a subclass of `Exception` in Python — backoff code using `except Exception:` silently misses wrapper timeouts. Must catch `BaseException` or `CancelledError` explicitly
- When fixing retry/backoff, the gating must happen at the **outermost loop** that controls re-invocation, not inside the function being retried
- Akamai cooldown after thrashing can last 4+ hours — patience + reduced retry pressure is the only viable strategy
- 8/8 Webshare proxies for TabTouch are dead (HTTP 000) — direct mode masks this but CloudFront-fallback path is broken

**Action Items:**
- TAB long-term fix: Build Sportsbet-style CookieMinter (Playwright harvests Akamai cookies from www.tab.com.au → curl_cffi uses them for API calls) — ~half day focused session
