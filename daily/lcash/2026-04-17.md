- Added per-line tightening: Threes/Steals at line ≤5 max 0.5 gap; high-count props max 15% of line (was 30%)

- **3-way devig bug root cause:** `draw_key` was only constructed from `_over` keys (`key.endswith("_over")`). - **Fix required voiding stored picks:** deploying the code fix only prevents new phantom picks — old ones persist in Supabase and must be manually voided
- **Alt-line interpolation is unreliable:** when sharp and soft books have very different lines (e.g., Threes 0.5 vs 6.5), interpolation across that gap produces nonsensical true probabilities
