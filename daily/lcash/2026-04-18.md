### Session (11:48) [value-betting-scanner]

**Context:** Debugging and fixing phantom EV bugs in the value betting scanner (Pinnacle devig + interpolation)

**Key Exchanges:**
- Traced soccer moneyline phantom EV to a specific code path: `_under` keys couldn't detect the draw market (line 563 only built `draw_key` from `_over` keys), causing fallthrough to 2-way devig
- 2-way devig on a 3-way market absorbs ~27% draw probability into both sides, inflating Motor Lublin from true 3.76 odds to phantom 2.77 odds (+18.6% fake EV vs actual -12.7%)
- Discovered NBA player prop interpolation was bridging a 4-point line gap (Pinnacle 5.5 → Kalshi 9.5), producing 15.6% true prob when Kalshi's own price said 7.5%
- Data showed all books cluster at line 5.5 — no intermediate lines available to build a curve

**Decisions Made:**
- Use **Poisson model** (not logit) for count props across any line gap — produces 4.2% true prob at 9.5 vs logit's 15.6%
- Added **cross-validation gate**: if soft book's own devigged price disagrees with interpolation by >5pp, skip the pick
- Kept 50% EV hard cap + tightened line gap as additional safety layers
- Voided 31 phantom soccer moneyline picks from Supabase (all showing 15-42% fake EV)

**Lessons Learned:**
- Stored picks in Supabase persist after code fixes — must void old phantom picks manually, not just fix the generation path
- 2-way devig on 3-way markets is a catastrophic error — redistributes draw probability and creates massive phantom edges
- Prediction market books (Kalshi, Polymarket) offer fundamentally different line structures than traditional sportsbooks — deep alt lines vs main lines
- Cross-validating interpolated true prob against the soft book's own devigged odds is a powerful sanity check

**Action Items:**
- Monitor new picks post-deploy to confirm Poisson + cross-validation gate working correctly
- Watch for edge cases where cross-validation 5pp threshold might be too tight or too loose
- Resolved pick data still thin (110 picks) — need 400+ before drawing statistical conclusions on profitability
