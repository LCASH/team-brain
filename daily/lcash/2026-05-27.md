### Session (08:59) [server]

**Context:** Designing and testing resilient proxy architecture for Sportsbet scraping through Akamai anti-bot protection

**Key Exchanges:**
- Empirically confirmed Akamai's "two-door" architecture: Door 1 (IP/ASN gate) blocks before Door 2 (cookies/WAF) ever runs — cookies minted on residential IPs are useless when sent from datacenter IPs because the 397b deny fires at the connection layer
- Tested residential-mint → datacenter-push strategy: **definitively doesn't work** (403/397b on both `/racing-schedule` and `/push` via DC, even with fresh residential cookies)
- Discovered Akamai's denylist is **behavioural, not ASN-based** — Mac on Superloop (AS38195, legit AU consumer ISP) got banned after a day of testing, same as datacenter IPs
- Acquired 20 Webshare static residential IPs (AU, AS3356 Level 3) — all 20 pass Akamai with HTTP 200
- Confirmed **cookies are portable across residential IPs** in the same Webshare pool — mint on IP A, use on IP B = 200 + valid CLEVS body

**Decisions Made:**
- Architecture locked: single Playwright cookie minter (rotating exit IP per refresh) + 20-way curl_cffi PushClient fan-out with per-IP sticky binding (`race_id % 20`)
- Per-IP rate cap: ~360 polls/hr with 1.5-3s jitter (vs Eve's ~80,000/hr that burned in 6 hours)
- All traffic must originate from residential IPs — no datacenter shortcut possible
- Hold off probing from Mac home IP and Eve; let them age 7-30 days off Akamai's behavioural list

**Lessons Learned:**
- **Don't probe from any IP you care about** until rate limiting is deployed — each test request costs IP reputation, and we burned 3 IPs in 24 hours of testing alone
- Akamai's 397b deny page is served at the TCP/IP layer before request headers/cookies are even inspected — no cookie, user-agent, or TLS trick can bridge the IP-class gap
- Eve running unrestricted for 6 hours = burned; rate limiting must be in place from request 1 on any new IP, not "deploy then tune"
- Static residential IPs on Tier-1 ISP ASNs (Level 3/Lumen) pass Akamai cleanly — the ASN matters for Door 1
