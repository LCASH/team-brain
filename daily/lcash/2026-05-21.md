**Decisions Made:**

**Lessons Learned:**
- **Cloudflare bot detection is session-aware, not just rate-aware.** Stateless round-robin at 10/sec from one IP → banned after 90 reqs. Same IP with persistent cookie jar at 5/sec → 1500/1500 clean. The `__cf_bm` cookie warmup makes Cloudflare treat proxy traffic like a real user.
- **Residential IP isn't unlimited either** — Dell's residential IP got 1015'd after cumulative load from Mac tests + smoke test. The "residential = free pass" assumption was wrong.
- **Sticky proxy architecture:** deterministic hash of eventId → fixed proxy, each proxy has own aiohttp session + cookie jar, auto-refresh sessions every 25 min (before `__cf_bm` 30-min TTL), 429 → quarantine 2 min + auto-reassign to next proxy.
