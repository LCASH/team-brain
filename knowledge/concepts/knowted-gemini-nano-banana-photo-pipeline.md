# Knowted Gemini Nano Banana AI Photo Pipeline

On 2026-06-16, lcash integrated AI-generated lifestyle photography into the Knowted ad creative pipeline using Gemini 3 Pro Image (Nano Banana Pro). The prior Puppeteer HTML→PNG pipeline (see [[concepts/html-css-ad-creative-renderer]]) produced pixel-perfect text cards but lacked photographic/lifestyle imagery. The Nano Banana integration generates on-brand lifestyle photos (meeting rooms, laptop setups, team collaborations) that serve as full-bleed backgrounds in a new `photoStatement` template. A critical limitation: AI-generated on-screen UI text is garbled at small sizes — the solution is gradient scrim overlays to hide AI text artifacts, with real text baked on top via the existing HTML/CSS pipeline.

## Key Points

- **Gemini 3 Pro Image (Nano Banana Pro)** runs through existing `gemini.js` on the free `generateContent` API path — no MCP server setup needed; API key confirmed working for image generation
- **`photoStatement` template**: Full-bleed AI photo + gradient scrim + headline/CTA overlaid via HTML/CSS — combines AI photo backgrounds with pixel-perfect text rendering
- **AI text rendering is garbled**: On-screen UI text in AI-generated images is nonsensical at small sizes — a known limitation of all current diffusion models; gradient scrim hides the affected areas
- **Brand green threads through generations**: When Knowted's brand green (#00ff88) is specified in prompts, it appears in walls, knitwear, UI elements — a useful but sometimes excessive effect
- **macOS sandbox blocks `~/Downloads/`**: CLI tools can't read from Downloads; nano-banana skill folder was copied into the project directory as a workaround
- **Assets are gitignored** (regenerable from committed prompts/code); user can force-add approved images if desired
- **4-week posting rollout plan**: 70/20/10 organic schedule (70% educational/value, 20% product, 10% promotional), one content lane per avatar

## Details

### Integration Architecture

The Nano Banana skill (10 files) provides a conversational interface for image generation via the Gemini API. Rather than setting up the MCP server (which the skill assumes), lcash routed requests through the existing `gemini.js` wrapper that was already proven during the Google AI free-tier investigation in [[concepts/html-css-ad-creative-renderer]]. The free-tier image generation quota issue from 2026-06-12 was resolved — the API key now supports `gemini-3-pro-image` on the `generateContent` path.

The pipeline flow:
1. **Generate**: Gemini produces lifestyle photos from text prompts (meeting scenes, laptop close-ups, team discussions)
2. **QA**: Human reviews for brand alignment — on-brand results but UI text garbled (expected)
3. **Composite**: `photoStatement` template overlays gradient scrim + headline + CTA text via Puppeteer HTML rendering
4. **Output**: 3 sizes per concept (1080×1080 feed, 1080×1350 stories, 1200×628 LinkedIn)

### The Text Rendering Limitation

All current diffusion models (Gemini, Midjourney, DALL-E, Flux, Stable Diffusion) struggle with text rendering in generated images. Small text on screens, signs, or documents in AI photos appears as garbled pseudo-text. For Knowted's use case — where ads feature laptops and screens — this means any "product screenshot" generated by AI will show nonsensical text on the screen.

The workaround architecture is the same hybrid approach established in [[concepts/html-css-ad-creative-renderer]]: AI generates the photographic elements (people, rooms, devices, lighting), and the HTML/CSS pipeline composites real text on top. The gradient scrim (a semi-transparent dark gradient overlaid on the bottom third of the photo) serves dual purpose: hiding garbled AI text in that region AND providing contrast for the overlaid headline text.

### Content Strategy Integration

The AI photo pipeline feeds into a 4-week content rollout plan with a 70/20/10 ratio:
- **70% educational/value**: Tips, meeting productivity insights, "did you know" posts using the educational `statement` template
- **20% product**: Feature highlights and demos using `photoStatement` with AI lifestyle backgrounds
- **10% promotional**: Competitive comparisons using the existing `comparison` and `vsLogo` templates

Each of the four ICP personas (see [[concepts/knowted-icp-land-expand-personas]]) gets its own content lane: solopreneurs see "solo meeting prep" imagery, students see "lecture capture" imagery, corporate callers see "team meeting" imagery.
